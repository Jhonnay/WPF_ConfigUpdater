Miniserver Updater:

You can update and maintain your Miniservers installations easily with the Miniserver Updater.

Features:

  • Overview about a Miniserver
  • Version
  • Project name
  • Configuration Type, i.e. Standalone or Client Gateway
  • Updatelevel
  • Updating
  • Via Config one by one
  • Set Language of connecting/updating Miniserver
  • Via AutoUpdate if standalone
  • Managing the Miniserver list
  • Add Miniserver
  • Delete Miniserver
  • Edit entry
  • Easy access to:
  • Webinterface local or remote
  • Copy SNR
  • Copy Password
  • FTP via File Explorer
  • LPH of the Miniserver
  • Crashlog Server
  • Changing Updatelevel
  • Connecting to Miniserver
  • Will start the Config with specified Language
  • Will load from Miniserver
  • Save / Load different Miniserver Configuration Lists via JSON
  • Application Settings
  • Default Config Path
  • Default Miniserver configuration path
  • Loxone app
  • Open Loxone App
  • Kill Loxone App
  • Open Loxone App in Debug

Updating

Updates are not possible if the Miniserver is on a higher Version. The local IP address is used if it is defined. The language is used for the config if defined, otherwise English US is used. There is a timeout of 17 minutes. Config remains open if this happens.

Connect Config

Connects to the Miniserver and loads the current program. The local IP address is used if it is defined. The language is used for the config if defined, otherwise English US is used.

Application Settings

These Settings are used when the program is launched.

  • Miniserver Configuration Path: This list will be loaded on startup
  • Default Config Path: This EXE will used on startup
